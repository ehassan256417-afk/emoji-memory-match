# Emoji Memory Match
**Date:** 2026-04-06  
**Objective:** Match all emoji pairs as fast as possible.  
**Rules:** Flip two cards at a time. Matching pairs increase your score.  
**Tech:** HTML5, Bootstrap 5, CSS3, ES Modules, Web Storage

**Code Snippet Example:**
```javascript
// Shuffle cards
function shuffle(array) {
    return array.sort(() => Math.random() - 0.5);
}
